### ISaleEntityNFT

| Param      | Type     | Required | Description                                                 |
|------------|----------|----------|-------------------------------------------------------------|
| `contract` | `String` | ✅        | Address of NFT                                              |
| `id`       | `String` | ✅        | ID of NFT                                                   |
| `amount`   | `Number` | ✅        | Amount of NFTs sold. Hardcode it as `1` for `ERC721` sales. |
